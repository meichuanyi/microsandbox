## Sandbox methods

### snapshot()

Capture full VM state (memory, CPU registers, filesystem). Pauses the sandbox. Returns `Snapshot`.

### resume()

Resume the sandbox after a snapshot.

---

## Snapshot (instance)

### save_named(name) / saveNamed(name)

Persist this snapshot under a name so it survives beyond the original sandbox's lifetime.

### restore(sandbox_name)

Fork a new sandbox from this snapshot. Skips boot and setup entirely. Returns `Sandbox`.

### restore_with(name, options)

Fork with config overrides (memory, cpus, env). Returns `Sandbox`.

---

## Snapshot (static)

### Snapshot.get(name)

Load a named snapshot. Returns `Snapshot`.

### Snapshot.list()

List all saved snapshots. Returns `Vec<SnapshotInfo>`.

### Snapshot.delete(name)

Delete a named snapshot.
