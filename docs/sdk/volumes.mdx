## Volume.builder(name) / Volume.create(options)

Create a named volume. Options: `quota(mib)` / `quotaMib`.

## Volume.get(name)

Get a handle to an existing named volume. Returns `VolumeHandle`.

## Volume.list()

List all named volumes. Returns `Vec<VolumeInfo>`.

## Volume.remove(name)

Delete a named volume.

## VolumeHandle

- `name()` — volume name
- `fs()` — host-side filesystem access (read/write files without a running sandbox)
- `remove()` — delete this volume

## Mount types

Used in `SandboxBuilder.volume()` / `SandboxConfig.volumes`:

### Mount.bind(host_path, options?)

Mount a host directory into the sandbox. Options: `readonly`.

### Mount.named(volume_name, options?)

Mount a named volume. Options: `readonly`.

### Mount.tmpfs(options?)

In-memory filesystem, discarded when the sandbox stops. Options: `sizeMib`.
