## SecretBuilder (Rust)

Used in `SandboxBuilder.secret(|s| s...)`:

### env(var_name)

Environment variable to expose the placeholder as. Required.

### value(secret_value)

The real secret value. Required.

### placeholder(string)

Custom placeholder string. Default: `$MSB_<env_var>`.

### allow_host(hostname)

Add an allowed host (exact match).

### allow_host_pattern(pattern)

Add an allowed host with wildcards (e.g., `*.openai.com`).

### inject_headers(bool)

Replace in HTTP headers. Default: `true`.

### inject_basic_auth(bool)

Replace in Authorization headers. Default: `true`.

### inject_query(bool)

Replace in URL query string. Default: `false`.

### inject_body(bool)

Replace in HTTP body. Default: `false`.

### require_tls_identity(bool)

Require TLS interception before substituting. Default: `true`.

## Secret.env (TypeScript)

```typescript
Secret.env(envVar, {
    value: string,
    allowHosts?: string[],
    allowHostPatterns?: string[],
    injectHeaders?: boolean,    // default: true
    injectBasicAuth?: boolean,  // default: true
    injectQuery?: boolean,      // default: false
    injectBody?: boolean,       // default: false
})
```

## Shorthand

`secret_env(env, value, host)` — equivalent to `.secret(|s| s.env(env).value(value).allow_host(host))`.

## Violation actions

Configured via `NetworkBuilder.on_secret_violation()`:

- **`Block`** — silently drop the request (default)
- **`BlockAndLog`** — drop and log a warning
- **`BlockAndTerminate`** — drop and shut down the sandbox
