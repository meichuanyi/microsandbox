See [Filesystem](/sandboxes/filesystem) for usage examples and extensible backends.

All methods are on `SandboxFs`, obtained via `sb.fs()`. Uses the same channel as command execution — no SSH, no network.

## write(path, content)

Write content to a file in the sandbox.

## read(path)

Read a file as bytes. Returns `Vec<u8>` / `Buffer`.

## read_to_string(path) / readString(path)

Read a file as a UTF-8 string.

## read_stream(path) / readStream(path)

Stream a file in ~3 MiB chunks. Returns `FsReadStream`.

## list(path)

List directory entries. Returns `Vec<FsEntry>`.

## stat(path)

Get file metadata. Returns `FsEntry`.

## exists(path)

Check if a path exists. Returns `bool`.

## copy_from_host(host_path, guest_path) / copyFromHost(hostPath, guestPath)

Copy a file from the host machine into the sandbox.

---

## FsEntry

- `path: String` — file path
- `kind: FsEntryKind` — `file`, `directory`, or `symlink`
- `size: u64` — file size in bytes

## FsReadStream

- `recv()` — receive next chunk (`Buffer` / `None` at end)
- `[Symbol.asyncIterator]` — use with `for await...of`
