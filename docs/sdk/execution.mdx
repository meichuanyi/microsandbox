## exec(cmd, args?)

Run a command and collect stdout, stderr, and the exit code once it completes. Returns `ExecOutput`.

## exec_with(cmd, options) / execWithConfig(options)

Run a command with per-execution overrides: working directory, environment variables, timeout, and POSIX resource limits. Returns `ExecOutput`.

## shell(script)

Run a command through the sandbox's configured shell (defaults to `/bin/sh`). Returns `ExecOutput`.

## exec_stream(cmd, args?) / execStream(cmd, args?)

Run a command with streaming output. Returns `ExecHandle` that emits `Stdout`, `Stderr`, and `Exited` events as they happen.

## exec_stream_with(cmd, options)

Streaming execution with per-execution overrides. Enable `stdin_pipe` to write to the process, and `tty` to allocate a PTY. Returns `ExecHandle`.

## shell_stream(script) / shellStream(script)

Shell command with streaming output. Returns `ExecHandle`.

## attach(cmd?) / attach_shell()

Bridge your terminal to a fully interactive PTY session inside the sandbox. Returns the exit code. Press `Ctrl+]` (or configured detach keys) to disconnect without stopping the process.

## attach_with(cmd, options) / attachWithConfig(options)

Interactive attach with options: env, cwd, detach keys. Returns the exit code.

## ExecOutput

- `code: i32` — exit code
- `success: bool` — `true` if exit code is 0
- `stdout(): String` — collected stdout
- `stderr(): String` — collected stderr

## ExecHandle

- `recv()` — receive next event (`Stdout`, `Stderr`, `Exited`)
- `take_stdin()` — get stdin writer (requires `stdin_pipe`)
- `wait()` — wait for process to exit
- `id()` — session ID

## ExecConfig

- `args: Vec<String>` — command arguments
- `cwd: String` — working directory
- `env: Map<String, String>` — environment variables
- `timeout: Duration` — execution timeout
- `rlimit: (Resource, u64)` — POSIX resource limit
- `stdin_pipe: bool` — enable stdin writer (default: `false`)
- `tty: bool` — allocate a PTY (default: `false`)
